CalDAV + CardDAV Manager

Built for teams that run their own sync stack

Manage CalDAV and CardDAV in one clean control center.

Davvy combines a modern web dashboard with a production-ready DAV endpoint, so admins and users can manage calendars, address books, sharing, and client compatibility from one place.

What ships by default

  • Default Personal Calendar and Contacts resources per user
  • Export support for ICS, VCF, and full ZIP bundles
  • Owner and admin sharing controls with per-resource permission enforcement
  • Contact workflows: moderation queue, milestone calendars, and Apple compatibility mirroring
